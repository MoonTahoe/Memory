A Game of Memory
================
This Javascript card game that will server as a tutorial for learning advanced Javascript concepts.

Topics
------
* Object Orientation
* Design Patterns
    * Observers
    * Iterator
    * Singletons
* BDD with Cucumber and Gherkin
* TDD with Mocha
* Grunt Publishing
* LESS and CSS, Responsive Design
* CD with Travis CI and Heroku
